# Python-Project-Weather-Analysis

**📌 Project Overview**

This project focuses on analysing weather data using Python to explore patterns, trends, and key insights. The analysis includes data cleaning, exploration, and visualisation to better understand weather conditions over time.

**🎯 Objectives**

Analyse weather conditions such as temperature, humidity, and wind speed

Identify patterns and trends in the dataset

Perform data cleaning and preprocessing

Create visualisations to present insights clearly

**📂 Dataset**

The dataset contains weather-related information such as:

Date/Time

Temperature

Humidity

Pressure

Wind Speed

Visibility

Weather Condition

**🛠 Tools & Technologies**

Python

Pandas (data manipulation)

NumPy (numerical operations)

Matplotlib / Seaborn (data visualisation)

Jupyter Notebook

**🔍 Process**

1. Data Cleaning & Preparation

  Handled missing or inconsistent values
  
  Converted data types where necessary
  
  Structured dataset for analysis

2. Exploratory Data Analysis (EDA)

  Analysed distribution of weather parameters
  
  Identified trends and relationships between variables
  
  Filtered data based on conditions

3. Data Visualisation

  Created graphs to show:
  
  Temperature trends
  
  Humidity levels
  
  Weather condition distribution
  
  Used visualisations to make insights easy to understand

**📈 Key Insights**

Identified variations in temperature over time

Observed patterns in humidity and weather conditions

Analysed how different weather factors relate to each other
